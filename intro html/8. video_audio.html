Audio

Instructions for Audio:

Here is some attribute of Audio elements

  1. controls: Add controls like play pause
  2. autoplay: Autoplay when page loads
  3. loop: Repeat the video
  4. mute: Mute by default
  5. preload: none,metadata,auto
    • auto: download auto while loading. none: nothing metadata: only load meta information

Video

Instructions for Video:

Here is some attribute of video elements

  1. controls: Add controls like play pause
  2. autoplay: Autoplay when page loads
  3. loop: Repeat the video
  4. mute: Mute by default
  5. poster: specifies the image before the video starts playing
